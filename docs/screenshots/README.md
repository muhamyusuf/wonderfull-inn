# Screenshots Directory

This directory contains application screenshots for documentation.

## Required Screenshots

1. **landing.png** - Homepage with hero section and featured packages
2. **packages.png** - Package browsing page with filters
3. **booking.png** - Booking form with date picker and traveler selection
4. **agent-dashboard.png** - Agent dashboard with stats and booking management
5. **tourist-dashboard.png** - Tourist dashboard with booking overview
6. **reviews.png** - Review system showing rating and comments

## How to Capture Screenshots

1. Run the development server: `bun run dev`
2. Navigate to each page
3. Use browser dev tools or screenshot tool
4. Recommended resolution: 1920x1080 (desktop) and 375x812 (mobile)
5. Save with the filenames listed above

## Tips

- Clear browser data for clean screenshots
- Use demo accounts for consistent data
- Capture both desktop and mobile views
- Ensure no console errors visible
- Use light mode for better visibility (or consistent theme)
